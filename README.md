# Lemmehand
a Darn Fast Proxy Server, written in Crystal

# Usage
1. `git clone https://github.com/codenoid/lemmehand.git`
2. `cd lemmehand`
3. `crystal build server.cr`
4. `./server --port 80`
5. Test !, go to browser or use http client, http://example.com/?get=https://webtogo.com
